## Feature Overview

Steve's Simple Storage is a multiblock storage system that grows with you. Start with a single Storage Box holding
10,000 items and expand all the way up to 10,240,000 items with an Ultimate Storage Box.

**Core features:**

- Massive item storage in a single system
- Right-click the Storage Core to access all items instantly
- Search, sort, and filter with Search Box and Sort Box
- Craft directly from storage with Crafting Box and JEI integration
- Lock your storage with Security Box so only you (or people you allow) can access it
- Automate item movement with Input, Extract, and Eject Ports
- Share access remotely with Access Terminal
