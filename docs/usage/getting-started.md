# Getting Started

Build your first storage system in under 5 minutes.

## What You Need

The only required block is the **Storage Core**. Everything else is optional.

| Item | Purpose | Required? |
|------|---------|-----------|
| Storage Core | The brain of your system — right-click to access storage | Yes |
| Storage Box | Adds +10,000 item capacity to your system | Recommended |
| Blank Box | Cosmetic-only block for filling gaps in your multiblock | No |

## Step 1 — Place a Storage Core

Place a **Storage Core** block wherever you want your storage system to live. This is the only required block.

The Storage Core will automatically detect any adjacent Storage Box blocks and add their capacity to your total.

## Step 2 — Add Storage Boxes (Optional)

Place **Storage Box** blocks adjacent to the Storage Core (touching on any face). Each one adds capacity:

| Block | Capacity |
|-------|----------|
| Storage Box (Basic) | +10,000 |
| Condensed Storage Box | +40,000 |
| Compressed Storage Box | +80,000 |
| Super Storage Box | +160,000 |
| Ultra Storage Box | +640,000 |
| Hyper Storage Box | +2,560,000 |
| Ultimate Storage Box | +10,240,000 |

Storage boxes must touch the Storage Core directly or indirectly through other storage boxes. The system expands outward in all directions.

## Step 3 — Open the GUI

**Right-click** the Storage Core to open the storage GUI. You will see:
- **54 storage slots** (6 rows × 9 columns) for your items
- **Your player inventory** at the bottom

The top of the GUI shows your current item count and total capacity (e.g., `1,234 / 10,000`).

## Step 4 — Insert Items

There are two ways to insert items:

**Drag and drop:** Drag items from your player inventory into any storage slot.

**Shift-click:** Hold **Shift** and click an item in your player inventory. It will automatically move into the first available storage slot.

Items of the same type stack automatically — you don't need to organize them manually.

## Step 5 — Retrieve Items

**Click** a storage slot to pick up items:
- **Left-click** — pick up a full stack
- **Right-click** — pick up half a stack

**Shift-click** a storage slot — move the entire stack to your player inventory at once.

Items are retrieved in the order they appear in the GUI. Use the **Sort Box** to reorder them.

## Step 6 — Expand Your System

As your storage fills up, add more blocks:

1. Place additional Storage Box blocks adjacent to your existing multiblock
2. Open the Storage Core GUI — the capacity updates automatically
3. For even more capacity, upgrade to Condensed, Compressed, or higher-tier boxes

## Optional: Add Feature Blocks

Place these blocks adjacent to your Storage Core to unlock extra features:

| Block | What it Does |
|-------|-------------|
| **Crafting Box** | Adds a 3×3 crafting grid connected to your storage. Shift-click a recipe in JEI to auto-pull ingredients from storage and craft. |
| **Search Box** | Adds a search bar to the top of the storage GUI. Type to filter items in real time. |
| **Sort Box** | Adds a sort button to the storage GUI. Click it to reorder items. |
| **Statistics Box** | Adds a detailed metrics display (item types, most common item, etc.) to the storage GUI. |
| **Security Box** | Lets you lock the storage system so only you (or people you whitelist) can access it. |
| **Access Terminal** | Place up to 64 blocks away from the Storage Core — right-click to open the same GUI remotely. |

## Optional: Automate with Ports

For automation, place these blocks adjacent to the Storage Core:

| Block | What it Does |
|-------|-------------|
| **Input Port** | Automatically pulls items from the inventory in front of it into your storage. |
| **Extract Port** | Automatically extracts items from your storage into the inventory in front of it. |
| **Eject Port** | The opposite of an Input Port — pushes items out of an adjacent inventory. |
| **Storage Interface** | Exposes your storage to any pipe or item network mod (Mekanism, Refined Storage, Create, etc.). |

## Tips

- **Dolly + Key:** Use a **Dolly** to pick up and move your entire multiblock (core + all attached blocks) without breaking it. Lock it with a **Key** first so nobody else can break it.
- **Count display:** Large item counts are shown with suffixes — 10,000 items shows as `10.0K`, 1,000,000 shows as `1.0M`.
- **Advancements:** As you build larger storage systems, you'll unlock 21 different advancements tracking your progress.
